# Ortho-Analytic Landing Page

## Requirements

You need **Node.js** installed. If you don't have it:
1. Go to https://nodejs.org
2. Download and install the **LTS** version
3. Restart your terminal after installing

---

## Setup & Run

Open your terminal, navigate to this folder, then run these commands one by one:

```bash
npm install
```

```bash
npm run dev
```

Then open your browser and go to:

```
http://localhost:5173
```

That's it. The page will be running locally on your laptop.

---

## Stop the server

Press `Ctrl + C` in the terminal.

---

## To run again later

You only need to run `npm install` once. Next time just:

```bash
npm run dev
```
